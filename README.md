- Command Line Arguments & I/O Operations
- Exception Handling
- Regular Expressions (Regex)
- List Comprehensions & Lambda Functions

### 📊 Data Science

- Introduction to Artificial Intelligence & Machine Learning
- Statistics for Data Science
